¿Qué es una API?

Una API como su nombre en inglés lo indica application programming interface o en español que significa interfaz de programación de aplicaciones, que quiere decir esto esto significa que una API es la parte de un codigo que hace que dos aplicaciones se comuniquen entre si.


Como sabes la comunicacion es transmitir informacion entre un emisor y un receptor esto quiere decir que dos aplicaciones se estaran transmitiendo informacion entre si, para hacerlo mas facil de entender te lo explicare asi. Digamos que tienes hambre y no quieres cocinar y tampoco salir de casa por comida asi que decides llamar a Mcdonals por ejemplo desde una aplicacion entonces para que me entiendas la informacion que necesitamos la tiene un servidor externo que en este caso seria Mcdonals y nuestra aplicacion que necesita la informacion seria nuestra casa entonces ¿Cual es la API en este ejemplo? la respuesta es sencilla la informacion seria la hamburguesa que pediste y la API seria el repartidor que llevara la hamburguesa (informacion) desde Mcdonals (servidor) hasta tu casa (tu computadora)


Bien ahora que ya sabes que es y como funciona una API te explicare que es un Deepracer
